class spectree.response.Response(*codes: str, **code_models: Union[Type[pydantic.main.BaseModel], None, Tuple[Optional[Type[pydantic.main.BaseModel]], str]])[source]

response object

Parameters

  • codes – list of HTTP status code, format(‘HTTP_[0-9]{3}’), ‘HTTP_200’

  • code_models – dict of <HTTP status code>: <pydantic.BaseModel> or None or a two element tuple of (<pydantic.BaseModel> or None) as the first item and a custom status code description string as the second item.

examples:

>>> from spectree.response import Response
>>> from pydantic import BaseModel
...
>>> class User(BaseModel):
...     id: int
...
>>> response = Response(HTTP_200)
>>> response = Response(HTTP_200=None)
>>> response = Response(HTTP_200=User)
>>> response = Response(HTTP_200=(User, "status code description"))
